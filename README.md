# Gesture Recognition Project 🤚✨

A Python project that detects and recognizes hand gestures in **real-time** using **OpenCV** and **NumPy**.  
Perfect for computer interaction, learning, or fun experiments with gesture control.

---

## 🚀 Features
- Real-time hand gesture detection via webcam  
- Supports multiple gestures  
- Modular and easy-to-extend code  

---

## 🛠️ Requirements
- Python 3.10
- OpenCV (`opencv-python`)  
- NumPy  
- Other dependencies listed in `requirements.txt`  

---

## ⚡ How to Run
1. **Clone this repository**
   git clone <https://github.com/codophilia/Gesture-Detection-Cursor.git>

2. **Install dependencies**
`pip install -r requirements.txt`

3. **Run the main script**
    `python gesture_main.py`
